## Formación real, no solo alfabetización

Nuestros cursos van más allá de lo básico. No se trata de un curso de computación tradicional, sino de preparar a nuestra gente con las habilidades que hoy abren puertas en cualquier parte del mundo.

<Info>
  Todos nuestros cursos parten desde cero, sin importar tu nivel de experiencia previo. Lo que importa es tu disposición para aprender.
</Info>

## Áreas de especialización

### Desarrollo Web
Aprende HTML, CSS, JavaScript y frameworks modernos para crear sitios y aplicaciones web profesionales.

**Proyectos típicos:**
- Sitio web personal o de portafolio
- Landing page para negocios locales
- Aplicación web interactiva
- E-commerce básico

### Programación
Desde los fundamentos de la lógica de programación hasta el desarrollo de software completo.

**Proyectos típicos:**
- Sistema de gestión para negocios
- Automatización de tareas
- Aplicaciones de escritorio
- Scripts y herramientas útiles

### Herramientas Profesionales
Domina Git, terminal, editores de código, y otras herramientas esenciales del desarrollo moderno.

**Habilidades clave:**
- Control de versiones con Git y GitHub
- Trabajo en línea de comandos
- Configuración de entornos de desarrollo
- Colaboración en equipos remotos
